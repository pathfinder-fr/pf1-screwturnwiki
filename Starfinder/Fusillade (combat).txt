''Vous vous servez de vos nombreux membres pour faire pleuvoir les projectiles sur vos ennemis.''

'''Catégorie ''': Combat

'''Conditions ''': bonus de base à l’attaque +1, quatre bras ou plus.

'''Avantage ''': lors d’une attaque à outrance quand vous maniez au moins quatre armes légères identiques, vous pouvez tirer avec toutes ces armes simultanément pour dupliquer les effets d’une arme automatique (voir page 180). Vous videz les chargeurs de toutes les armes légères avec lesquelles vous tirez et vous considérez que cette attaque est en mode automatique. Faites la somme de toutes les munitions tirées par l’ensemble de vos armes légères quand vous déterminez le nombre maximum de créatures que vous pouvez toucher.
