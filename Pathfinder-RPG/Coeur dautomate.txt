{s:BDTitre|Coeur d'automate|}
'''Aura''' [[Transmutation]] puissante ; '''NLS''' 20 ; '''Emplacement''' aucun ; '''Poids''' 500g
(((Description)))
Ces cylindres de métal luisants sont souvent plaqués or ou argent et décorés avec de minuscules symboles d'arcanes brillants. Ces cœurs sont principalement utilisés pour donner vie aux automates mais un cœur d'automate peut aussi être récolté sur un automate pour augmenter la puissance de sorts. Un lanceur de sort tenant un cœur d'automate considère son NLS comme plus haut de 4 niveaux.

Le pouvoir inné d'un cœur d'automate dépend du [[FP]] de l'automate dont il à été extrait : il contient un point de sort par FP. Par une [[action rapide]], le lanceur de sorts dépenser des niveaux de sorts du cœur d'automate pour lancer un sort qu'il a préparé sans utiliser un de ses emplacements de sorts quotidiens. Le nombre de points de sorts requis pour utiliser cette capacité est égal à 2 fois le niveau du sort lancé. Une fois que tous les points de sorts ont été dépensés du cœur d'automate, il ne peut plus être utilisé pour donner la vie à un automate. Il n'existe aucun moyen connu pour restaurer un cœur d'automate.

Un cœur d'automate peut aussi être utilisé pour réduire le coût de création d'une [[type Créature artificielle|créature artificielle]]. Quand il est inclus dans le processus de création d'une [[type Créature artificielle|créature artificielle]], un cœur d'automate est équivalent à 2500 po de composante par point de sort. Un cœur d'automate utilisé de cette façon peut être retiré en suivant les règles normales permettant de retirer un cœur d'automate.

Les automates convoitent ces cœurs, car ils peuvent les utiliser pour réactiver des automates alliés dont les cœurs ont lâché, ou ceux dont les cœurs ont été volés. Les automates sont connus pour traquer sans relâche et tuer toute créature qui détient un de ces objets.
(((Destruction)))
Un cœur d'automate peut seulement être détruit en le plaçant dans un cercle rituel composé de 6 autres cœurs d'automates. Ces cœurs doivent ensuite dépenser un nombre de points de sorts égal à ceux restant dans le cœur central. Cela surcharge le cœur, le détruisant pour toujours
