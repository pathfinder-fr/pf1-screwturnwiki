'''École''' [[divination]] ; '''Niveau''' [[alchimiste]] 1, [[barde]] 0, [[conjurateur]] 0, [[prêtre]] 0, [[druide]] 0,
[[inquisiteur]] 0, [[magus]] 0, [[paladin]] 1, [[rôdeur]] 1, [[ensorceleur]]/[[magicien]] 0, [[sorcière]] 0<br>
'''Durée''' 1 minute/[[niveau]]<br>
'''Jet de sauvegarde''' aucun ; '''Résistance à la magie''' non<br>
'''Restriction de cible''' ''[[Personnel (mot de pouvoir)|personnel]]''

La cible d’un sort incluant ce mot peut lire un texte écrit dans une langue particulière, dont la langue de la magie. Le mage choisit la langue quand il lance son sort et pendant toute sa durée, la cible peut lire cette langue comme s’il la maîtrisait couramment. S’il choisit la langue magique, la cible peut déchiffrer les parchemins, les glyphes et les autres écritures magiques sans les déclencher. Un sort incluant cet effet ne donne pas la capacité de connaître une langue en observant le texte ; le lanceur doit déjà la connaître ou avoir deviné dans quelle langue le document est écrit pour lancer son sort.<br>
:'''Augmentation :''' Ce mot effet peut utiliser le mot cible ''[[choisi]]''. Augmenter le mot de la sorte améliore son niveau de 1.
