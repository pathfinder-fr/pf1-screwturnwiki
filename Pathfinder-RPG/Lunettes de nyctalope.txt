<div class="BD">
{s:BDTitre|Lunettes de nyctalope|}
'''Aura''' [[Transmutation]] faible ; '''NLS''' 3 ; '''Emplacement''' [[Objets merveilleux portés sur les yeux|yeux]] ; '''Prix''' 12 000 po ; '''Poids''' —
(((Description)))
Les verres de ces lunettes ont été taillés dans un cristal noir. Même si elles paraissent opaques, quiconque les chausse y voit normalement et acquiert le pouvoir de [[vision dans le noir]] (jusqu’à une portée de 18 m). Les lunettes ne fonctionnent que si les deux verres sont présents.
(((Création)))
'''Conditions''' [[Création dobjets merveilleux|Création d'objets merveilleux]], ''[[sort vision dans le noir|vision dans le noir]]'' ; '''Coût''' 6 000 po
