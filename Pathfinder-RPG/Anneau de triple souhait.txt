{s:BDTitre|Anneau de triple souhait|{s:AE}}
'''Aura''' [[Évocation]] ou [[Universelle]] puissante (si ''[[miracle]]'' est utilisé) ; '''NLS''' 20 ; '''Emplacement''' anneau ; '''Prix''' 120 000 po ; '''Poids''' —
(((Description)))
Cet anneau est serti de trois rubis. Chacun d’entre eux stocke un sort de ''[[souhait]]'' auquel le porteur peut faire appel quand il le souhaite. Dès qu’un ''[[souhait]]'' est utilisé, le rubis correspondant disparaît. Si l’anneau est déterminé aléatoirement, lancez 1d3 afin de déterminer le nombre de rubis restants. Une fois tous les ''[[souhait|souhaits]]'' utilisés, l’anneau devient non magique.
(((Fabrication)))
'''Conditions''' [[Création danneaux magiques|Création d'anneaux magiques]], soit ''[[souhait]]'', soit ''[[miracle]]'' ; '''Coût''' 97 500 po
