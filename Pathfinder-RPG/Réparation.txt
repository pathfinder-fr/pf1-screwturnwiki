'''École''' [[Transmutation]] ; '''Niveau''' [[ensorceleur|Ens]]/[[magicien|Mag]] 0, [[barde|Bard]] 0, [[druide|Dru]] 0, [[prêtre|Prê]] 0, [[conjurateur|Conj]] 0, [[sorcière|Sor]] 0, [[Chaman|Cham]] 0<br/>
'''Temps d'incantation''' 10 minutes<br/>
'''Composantes''' [[composantes|V, G]]<br/>
'''Portée''' 3 m<br/>
'''Cible''' 1 objet pesant jusqu’à 500 g/niveau<br/>
'''Durée''' [[instantanée]]<br/>
'''Jet de sauvegarde''' [[Volonté]], [[Présentation des sorts#JETSDESAUVEGARDE|annule]] ([[Présentation des sorts#JETSDESAUVEGARDE|inoffensif]], objet) ; '''Résistance à la magie''' oui ([[Présentation des sorts#JETSDESAUVEGARDE|inoffensif]], objet)

Ce sort permet de réparer les dommages superficiels causés aux objets et leur rend 1d4 [[points de vie]]. Dans le cas d’un objet [[brisé|cassé]], l’objet n’est plus affecté par cette condition néfaste si l’objet se retrouve à 50% de ses [[points de vie]] ou moins. Pour que ce sort fonctionne, le personnage doit disposer de tous les morceaux de l’objet. Le personnage peut utiliser ce sort pour réparer un objet magique cassé à condition que son [[NLS|niveau de lanceur de sorts]] soit égal ou supérieur à celui de l’objet. Il peut aussi réparer un objet magique détruit (à 0 point de vie ou moins) mais le sort ne lui rend pas ses aptitudes magiques. Ce sort n’affecte pas les créatures (pas même les [[type créature artificielle|créatures artificielles]]). Ce sort n’a aucun effet sur les objets pervertis ou transmutés mais il peut réparer les dégâts qu’ils ont subis.
