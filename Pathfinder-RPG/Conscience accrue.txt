{s:MCA}'''École''' [[Divination]] ; '''Niveau''' [[Alchimiste|Alch]] 1, [[Barde|Bard]] 1, [[Chaman|Cham]] 1, [[Druide|Dru]] 1, [[Hypnotiseur|Hyp]] 1, [[Inquisiteur|Inq]] 1, [[Ensorceleur|Ens]]/[[Magicien|Mag]] 1, [[Médium|Méd]] 1, [[Occultiste|Occ]] 1, [[Psychiste|Psy]] 1, [[Rôdeur|Rôd]] 1</br>
'''Temps d’incantation''' 1 [[action simple]] \
'''Composantes''' [[Composantes|V, M/FD]] (un grain de café) \
'''Portée''' personnelle \
'''Cible''' le lanceur de sorts \
'''Durée''' 10 minutes/[[niveau]] (T) 

Le personnage entre dans un état de conscience aiguë qui lui permet de repérer plus de choses de son environnement et de se rappeler les informations sans effort. Il bénéficie d’un bonus de compétence de +2 aux tests de [[Perception]] et à tous les tests de [[Connaissances]] auxquelles il est formé.

Si ce sort est actif alors que le personnage doit faire un test d’[[initiative]], il peut instantanément dissiper le sort pour bénéficier d’un bonus de +4 sur ce test.
