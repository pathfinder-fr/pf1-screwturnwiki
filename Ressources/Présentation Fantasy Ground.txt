====Les races====
[^https://www.dropbox.com/s/pkvnqtt3fplnabx/PFJDR%20fr%20-%20Regles%20-%20Races.mod|Lien vers le module].
*''Note : Aucune image n'est présente dans ce module, vous devez ajouter des images dans le répertoire "refimage" !
Pour se faire :
1- Renommez "PFJDR fr - Regles - Races.mod" en .zip
2- Ouvrir avec Winzip ou Winrar
3- Placez vos images nommées humain.jpg nain.jpg elfe.jpg demielfe.jpg demiorque.jpg gnome.jpg et halfelin.jpg
4- Sauvegardez et renommez le .zip en .mod
5- Relancez FG2, ouvrez le module pour vérifier si les liens fonctionnent bien.''
